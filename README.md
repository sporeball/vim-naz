# vim-naz

adds syntax highlighting to [naz](https://github.com/sporeball/naz) files in Vim.

### install
using Vim's native package support:
```
git clone https://github.com/sporeball/vim-naz.git ~/.vim/pack/sporeball/start/vim-naz
```
